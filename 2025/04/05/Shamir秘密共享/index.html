ж—Ҙжңҹпјҡ2025е№ҙ3жңҲ28ж—Ҙ

1. з§ҳеҜҶе…ұдә«

з§ҳеҜҶ $s$ пјҲйҖҡиҝҮжҹҗз§Қж–№жЎҲпјүиў«еҲҶдёә $n$ дёӘйғЁеҲҶ,жҜҸдёӘйғЁеҲҶз§°дёәд»Ҫйўқ(share),з”ұдёҖдёӘеҸӮдёҺиҖ…жҢҒжңүпјҢдҪҝеҫ—

  • з”ұ $k$ дёӘжҲ–еӨҡдәҺ $k$ дёӘеҸӮдёҺиҖ…жүҖжҢҒжңүзҡ„йғЁеҲҶдҝЎжҒҜеҸҜйҮҚжһ„ $s$
  • з”ұе°‘дәҺ $k$ дёӘеҸӮдёҺиҖ…жүҖжҢҒжңүзҡ„йғЁеҲҶдҝЎжҒҜеҲҷж— жі•йҮҚжһ„ $s$

з§°иҜҘж–№жЎҲдёә $(k,n)$ й—Ёйҷҗз§ҳеҜҶе…ұдә«ж–№жЎҲпјҢ$k$ з§°дёәй—ЁйҷҗеҖјгҖӮ

2. Shamirз§ҳеҜҶе…ұдә«пјҲShamir Secret Sharing SSSпјү

Shamirз§ҳеҜҶе…ұдә«пјҲShamirвҖҷs Secret Sharing, SSSпјү жҳҜдёҖз§ҚеҜҶз ҒеӯҰеҚҸи®®пјҢз”ұ Adi Shamir еңЁ 1979 е№ҙжҸҗеҮәгҖӮе®ғзҡ„ж ёеҝғжҖқжғіжҳҜе°ҶдёҖдёӘз§ҳеҜҶпјҲеҰӮеҜҶй’ҘжҲ–ж•Ҹж„ҹж•°жҚ®пјүеҲҶжҲҗеӨҡдёӘйғЁеҲҶпјҲз§°дёәвҖңд»ҪйўқвҖқжҲ–вҖңsharesвҖқпјүпјҢе№¶е°Ҷиҝҷдәӣд»ҪйўқеҲҶеҸ‘з»ҷеӨҡдёӘеҸӮдёҺиҖ…гҖӮеҸӘжңүиҫҫеҲ°дёҖе®ҡж•°йҮҸзҡ„д»ҪйўқпјҲз§°дёәвҖңйҳҲ(yu)еҖјвҖқпјүжүҚиғҪжҒўеӨҚеҮәеҺҹе§Ӣз§ҳеҜҶпјҢиҖҢе°‘дәҺйҳҲеҖјзҡ„д»Ҫйўқж— жі•жҸҗдҫӣд»»дҪ•е…ідәҺз§ҳеҜҶзҡ„дҝЎжҒҜгҖӮ

SSS жҳҜдёҖз§Қй—Ёйҷҗз§ҳеҜҶе…ұдә«ж–№жЎҲпјҢе№ҝжіӣеә”з”ЁдәҺеҲҶеёғејҸзі»з»ҹгҖҒеҜҶз ҒеӯҰгҖҒж•°жҚ®еӨҮд»Ҫе’Ңе®үе…ЁеӨҡж–№и®Ўз®—зӯүйўҶеҹҹгҖӮ

еүҚиЁҖ

еӨҡйЎ№ејҸдёҠзҡ„ $k$ дёӘзӮ№иғҪеӨҹе”ҜдёҖзЎ®е®ҡе°ҸдәҺжҲ–зӯүдәҺ $kвҲ’1$ ж¬Ўзҡ„еӨҡйЎ№ејҸдёҖиҲ¬ең°пјҢи®ҫ${(x_1,y_1),вҖҰ,(x_k,y_k)}$жҳҜе№ійқўдёҠ$k$дёӘдёҚеҗҢзҡ„зӮ№жһ„жҲҗзҡ„зӮ№йӣҶпјҢйӮЈд№ҲеңЁе№ійқўдёҠеӯҳеңЁе”ҜдёҖзҡ„$k-1$ж¬ЎеӨҡйЎ№ејҸ$f(x)=a_0+a_1x+\cdots +a_{k-1}x^{k-1}$йҖҡиҝҮиҝҷ$k$дёӘзӮ№пјҢе…¶дёӯ$a_0,a_1,вҖҰa_k-1$жҳҜйҡҸжңәж•°гҖӮ

зЎ®е®ҡеҘҪй—ЁйҷҗеҖј$k$е’ҢжҖ»еҖј$n$пјҢиӢҘжҠҠз§ҳеҜҶ $s$еҸ–еҒҡ$f(0)$ пјҢдёәжҜҸдёӘеҸӮдёҺиҖ…з”ҹжҲҗдёҖдёӘд»Ҫйўқ$(x_i, y_i)$пјҢе…¶дёӯ$y_i = f(x_i)$гҖӮ$x_i(i=1\cdots n)$дҪңдёәеҸӮдёҺж–№idпјҢ$n$дёӘд»ҪйўқеҸ–еҒҡ$(x_i,f(x_i))(i=1,вҖҰn)$пјҢйӮЈд№ҲеҲ©з”Ёе…¶дёӯд»»ж„Ҹ$k$дёӘд»ҪйўқеҸҜд»ҘйҮҚжһ„$f(x)$пјҢд»ҺиҖҢеҸҜд»Ҙеҫ—еҲ°з§ҳеҜҶ$s$гҖӮ

жһ„йҖ 

  • и®ҫ$GF(q)$дёәеӨ§зҙ ж•°$q$з”ҹжҲҗзҡ„жңүйҷҗеҹҹпјҢе…¶дёӯ$q\ge n+1$.

  • з§ҳеҜҶ$s$жҳҜ$GF(q)/{0}$дёҠеқҮеҢҖйҖүеҢәзҡ„йҡҸжңәж•°пјҢеҚі$s\in GF(q)/{0}$.

  • еңЁ$GF(q)$дёҠжһ„йҖ дёҖдёӘ$k-1$ж¬ЎеӨҡйЎ№ејҸ$f(x)=a_0+a_1x+\cdots +a_{k-1}x^{k-1}$пјҢе…¶дёӯпјҡ$a_0=sпјҢa_i\in GF(q)/{0}(i\ne 0)$

  • $n$дёӘеҸӮдёҺиҖ…$P_1,P_2вҖҰ,P_n$пјҢе…¶дёӯ$P_i$зҡ„д»Ҫйўқдёә$f(i)$гҖӮд»»ж„Ҹ$k$дёӘеҸӮдёҺиҖ…жғіиҰҒеҫ—еҲ°з§ҳеҜҶ$s$пјҢеҸҜдҪҝз”Ё

дҫӢеҰӮпјҡеҒҮе®ҡдёҖдёӘз§ҳеҜҶеҖј$s=45$пјҢиҰҒжұӮе°Ҷе…¶еҲҶз»ҷ10дёӘеҸӮдёҺж–№пјҢиҮіе°‘ 3 ж–№еҸӮдёҺи®Ўз®—жүҚиғҪеӨҹжҒўеӨҚеҮәеҺҹе§Ӣз§ҳеҜҶпјҢеҚі $n=10,k=3$гҖӮжһ„йҖ еҜ№еә”зҡ„еӨҡйЎ№ејҸ $f(x)=45+19x+13x^2$гҖӮеңЁжӯӨеӨҡйЎ№ејҸдёҠйҖүеҸ–иҝһз»ӯзҡ„ж•ҙж•°зӮ№дҪңдёәеҗ„дёӘеӯҗз§ҳеҜҶпјҡ

з§ҳеҜҶйҮҚе»ә

еҜ№дәҺ $y_i$ еҖјзҡ„д»»ж„Ҹ $k$ дёӘеӯҗйӣҶпјҢжҲ‘д»¬еҸҜд»ҘйҖҡиҝҮжҸ’еҖјжүҫеҲ° $f(x)$ зҡ„еӨҡйЎ№ејҸзі»ж•°пјҢз„¶еҗҺи®Ўз®— $s=f(0)$гҖӮеҸҰдёҖж–№йқўпјҢд»…зҹҘйҒ“еӯҗйӣҶдёӯзҡ„ $kвҲ’1$ дёӘе№¶дёҚи¶ід»Ҙи®Ўз®—еҮә $s$гҖӮ

еңЁз”ҹжҲҗз§ҳеҜҶд»Ҫйўқзҡ„ж—¶еҖҷпјҢе·Із»ҸзЎ®е®ҡйңҖиҰҒ$k$ дёӘз§ҳеҜҶд»ҪйўқжқҘжҒўеӨҚеҺҹе§Ӣз§ҳеҜҶгҖӮеҚіж №жҚ®$(x_1,y_1),(x_2,y_2),вҖҰ,(x_k,y_k)$ зӯүдёҖзі»еҲ—зӮ№жһ„йҖ еҮәеҺҹе§Ӣзҡ„еӨҡйЎ№ејҸ $f(x)$пјҢиҝӣиҖҢжұӮи§Јеҫ—еҲ°з§ҳеҜҶеҖј $s=f(0)=a_0$гҖӮ

з”ұLagrangeжҸ’еҖје…¬ејҸеҫ—пјҡз»ҷе®ҡ$n+1$дёӘж•°жҚ®зӮ№$(x_0, y_0), (x_1, y_1), \ldots, (x_n, y_n)$пјҢе…¶дёӯжүҖжңүзҡ„$x_i$дә’дёҚзӣёеҗҢпјҢжӢүж јжң—ж—ҘжҸ’еҖјеӨҡйЎ№ејҸ$f(x)$еҸҜд»ҘиЎЁзӨәдёәпјҡ

е…¶дёӯпјҢ$\ell_i(x)$жҳҜжӢүж јжң—ж—ҘеҹәеӨҡйЎ№ејҸпјҲд№ҹз§°дёәжҸ’еҖјеҹәеҮҪж•°пјүпјҢе®ҡд№үдёәпјҡ

еҹәеӨҡйЎ№ејҸзҡ„жҖ§иҙЁ

  • жҜҸдёӘеҹәеӨҡйЎ№ејҸ$\ell_i(x)$ж»Ўи¶іпјҡ

  • еӣ жӯӨпјҢ$P(x_i) = y_i$еҜ№жүҖжңү$i$жҲҗз«ӢпјҢзЎ®дҝқеӨҡйЎ№ејҸйҖҡиҝҮжүҖжңүз»ҷе®ҡзӮ№гҖӮ

  1. еҪ“$j = i$ж—¶пјҲеҚі$x=x_{i}$пјү еҹәеӨҡйЎ№ејҸ$\ell_i(x)$зҡ„е®ҡд№үдёәпјҡ$\ell_i(x) = \prod_{\substack{0 \leq j \leq n \ j \neq i}} \frac{x-x_{j}}{x_{i}-x_{j}}$гҖӮе°Ҷ$x=x_{i}$д»Је…Ҙпјҡ

    • еҲҶеӯҗйғЁеҲҶпјҡ$x_i - x_j$пјҲдёҺеҲҶжҜҚзӣёеҗҢпјүгҖӮ
    • еҲҶжҜҚйғЁеҲҶпјҡ$x_i - x_j$пјҲдёҺеҲҶеӯҗзӣёеҗҢпјүгҖӮ

    еӣ жӯӨпјҢжҜҸдёҖйЎ№зҡ„еҖјдёә$\frac{x_i - x_j}{x_i - x_j} = 1$пјҢиҝһд№ҳеҗҺеҫ—еҲ°пјҡ

    $\ell_i(x_i)=\prod_{j\neq i}1=1.$

2.еҪ“$j \neq i$ж—¶пјҲеҚі$x = x_k$дё”$k \neq i$пјү

и§ӮеҜҹ$\ell_i(x)$зҡ„иҝһд№ҳз§ҜеҪўејҸпјҡ

$\ell_i(x)=\frac{(x-x_0)}{(x_i-x_0)}\cdot\frac{(x-x_1)}{(x_i-x_1)}\cdots\frac{(x-x_{i-1})}{(x_i-x_{i-1})}\cdot\frac{(x-x_{i+1})}{(x_i-x_{i+1})}\cdots\frac{(x-x_n)}{(x_i-x_n)}$

еҪ“$x = x_k$пјҲ$k \neq i$пјүж—¶пјҢиҝһд№ҳз§Ҝдёӯеҝ…е®ҡеӯҳеңЁдёҖйЎ№еҲҶеӯҗдёә$x_k - x_k = 0$пјҲеҚіз¬¬$k$йЎ№пјүгҖӮеӣ жӯӨпјҡ

$\ell_i(x_k) = \text{пјҲе…¶д»–йЎ№пјү} \cdot \frac{x_k - x_k}{x_i - x_k} \cdot \text{пјҲе…¶д»–йЎ№пјү} = 0.$

зӣҙи§ӮзҗҶи§Ј

  • $\ell_i(x)$жҳҜдёҖдёӘвҖңејҖе…іеҮҪж•°вҖқпјҡе®ғеңЁ$x_i$еӨ„еҸ–еҖјдёә 1пјҢеңЁе…¶д»–ж•°жҚ®зӮ№$x_j$пјҲ$j \neq i$пјүеӨ„ејәеҲ¶дёә 0гҖӮ
  • жҸ’еҖјеӨҡйЎ№ејҸзҡ„жһ„йҖ пјҡйҖҡиҝҮе°ҶжҜҸдёӘ$y_i$дёҺеҜ№еә”зҡ„$\ell_i(x)$зӣёд№ҳеҗҺзӣёеҠ пјҲ$P(x) = \sum y_i \ell_i(x)$пјүпјҢеҸҜд»ҘзЎ®дҝқпјҡ
    • еңЁ$x = x_i$ж—¶пјҢеҸӘжңү$\ell_i(x_i) = 1$пјҢе…¶д»–еҹәеҮҪж•°еқҮдёә 0пјҢеӣ жӯӨ$P(x_i) = y_i$гҖӮ
    • еңЁе…¶д»–зӮ№$x_j$пјҲ$j \neq i$пјүпјҢ$\ell_i(x_j) = 0$зҡ„жңәеҲ¶йҒҝе…ҚдәҶдёҚеҗҢ$y_j$д№Ӣй—ҙзҡ„е№Іжү°гҖӮ

дҫӢеӯҗйӘҢиҜҒ

еҒҮи®ҫжңү 3 дёӘзӮ№$(x_0, x_1, x_2)$пјҢеҹәеҮҪж•°$\ell_1(x)$дёәпјҡ

  • еңЁ$x = x_1$ж—¶пјҡ
  • еңЁ$x = x_0$жҲ–$x = x_2$ж—¶пјҡ

дёәдәҶйҮҚе»әз§ҳеҜҶпјҢд»»ж„ҸйҖүеҸ–3дёӘеҸӮдёҺж–№зҡ„з§ҳеҜҶд»ҪйўқдҪңдёәиҫ“е…ҘгҖӮиҝҷйҮҢйҖүжӢ© id дёә 2пјҢ3пјҢ6зҡ„еҸӮдёҺж–№иҝӣиЎҢз§ҳеҜҶйҮҚе»әгҖӮ

$(x_0пјҢy_0)=(2,135);пјҲx_1пјҢy_1)=(3,219);пјҲx_2пјҢy_2)=(6,627);$
