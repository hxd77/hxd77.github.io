Diffie-HellmanпјҲиҝӘиҸІ-иө«е°”жӣјпјүжҳҜдёҖз§ҚеҜҶй’ҘдәӨжҚўеҚҸи®®пјҢз”ұ Whitfield Diffie е’Ң Martin Hellman еңЁ 1976 е№ҙжҸҗеҮәгҖӮе®ғжҳҜзҺ°д»ЈеҜҶз ҒеӯҰдёӯжңҖйҮҚиҰҒзҡ„еҚҸи®®д№ӢдёҖпјҢз”ЁдәҺеңЁдёҚе®үе…Ёзҡ„йҖҡдҝЎжё йҒ“дёҠе®үе…Ёең°дәӨжҚўеҜҶй’ҘгҖӮDiffie-Hellman еҚҸи®®зҡ„ж ёеҝғжҖқжғіжҳҜе…Ғи®ёеҸҢж–№йҖҡиҝҮе…¬ејҖзҡ„йҖҡдҝЎжё йҒ“з”ҹжҲҗдёҖдёӘе…ұдә«зҡ„еҜҶй’ҘпјҢиҖҢж— йңҖдәӢе…Ҳе…ұдә«д»»дҪ•з§ҳеҜҶдҝЎжҒҜгҖӮ

Diffie-Hellman зҡ„ж ёеҝғжҰӮеҝө

  1. еҜҶй’ҘдәӨжҚўзҡ„зӣ®ж Үпјҡ
    • йҖҡдҝЎеҸҢж–№пјҲдҫӢеҰӮ Alice е’Ң BobпјүеёҢжңӣз”ҹжҲҗдёҖдёӘе…ұдә«зҡ„еҜҶй’ҘпјҢз”ЁдәҺеҗҺз»ӯзҡ„еҠ еҜҶйҖҡдҝЎпјҲеҰӮеҜ№з§°еҠ еҜҶпјүгҖӮ
    • иҝҷдёӘеҜҶй’ҘйңҖиҰҒеңЁе…¬ејҖзҡ„йҖҡдҝЎжё йҒ“дёҠз”ҹжҲҗпјҢеҚідҪҝж”»еҮ»иҖ…зӣ‘еҗ¬дәҶйҖҡдҝЎеҶ…е®№пјҢд№ҹж— жі•жҺЁеҜјеҮәе…ұдә«еҜҶй’ҘгҖӮ
  2. ж•°еӯҰеҹәзЎҖпјҡ
    • Diffie-Hellman еҹәдәҺзҰ»ж•ЈеҜ№ж•°й—®йўҳзҡ„ж•°еӯҰйҡҫйўҳгҖӮ
    • е®ғдҪҝз”ЁжЁЎе№Ӯиҝҗз®—е’ҢеӨ§зҙ ж•°зҡ„жҖ§иҙЁжқҘзЎ®дҝқе®үе…ЁжҖ§гҖӮ
  3. еҚҸи®®жөҒзЁӢпјҡ
    • еҸҢж–№еҚҸе•ҶдёӨдёӘе…¬ејҖзҡ„еҸӮж•°пјҡдёҖдёӘеӨ§зҙ ж•°$p$е’ҢдёҖдёӘз”ҹжҲҗе…ғ$g$пјҲ$g$жҳҜжЁЎ$p$зҡ„дёҖдёӘеҺҹж №пјүгҖӮ
    • еҸҢж–№еҗ„иҮӘйҖүжӢ©дёҖдёӘз§ҒеҜҶзҡ„йҡҸжңәж•°пјҲ$a$е’Ң$b$пјүпјҢе№¶и®Ўз®—иҮӘе·ұзҡ„е…¬й’Ҙпјҡ
      • Alice зҡ„е…¬й’Ҙпјҡ$A = g^a \mod p$
      • Bob зҡ„е…¬й’Ҙпјҡ$B = g^b \mod p$
    • еҸҢж–№дәӨжҚўе…¬й’ҘгҖӮ
    • еҸҢж–№дҪҝз”ЁеҜ№ж–№зҡ„е…¬й’Ҙе’ҢиҮӘе·ұзҡ„з§Ғй’Ҙи®Ўз®—е…ұдә«еҜҶй’Ҙпјҡ
      • Alice и®Ўз®—пјҡ$S = B^a \mod p$
      • Bob и®Ўз®—пјҡ$S = A^b \mod p$
    • з”ұдәҺ$S = g^{ab} \mod p$пјҢеҸҢж–№жңҖз»Ҳеҫ—еҲ°зӣёеҗҢзҡ„е…ұдә«еҜҶй’ҘгҖӮ

Diffie-Hellman зҡ„зӨәдҫӢ

  1. еҸӮж•°йҖүжӢ©пјҡ
    • йҖүжӢ©дёҖдёӘеӨ§зҙ ж•°$p = 23$е’ҢдёҖдёӘз”ҹжҲҗе…ғ$g = 5$гҖӮ
  2. з§Ғй’ҘйҖүжӢ©пјҡ
    • Alice йҖүжӢ©дёҖдёӘз§Ғй’Ҙ$a = 6$пјҢи®Ўз®—е…¬й’Ҙ$A = 5^6 \mod 23 = 8$гҖӮ
    • Bob йҖүжӢ©дёҖдёӘз§Ғй’Ҙ$b = 15$пјҢи®Ўз®—е…¬й’Ҙ$B = 5^{15} \mod 23 = 19$гҖӮ
  3. дәӨжҚўе…¬й’Ҙпјҡ
    • Alice еҸ‘йҖҒ$A = 8$з»ҷ BobгҖӮ
    • Bob еҸ‘йҖҒ$B = 19$з»ҷ AliceгҖӮ
  4. и®Ўз®—е…ұдә«еҜҶй’Ҙпјҡ
    • Alice и®Ўз®—$S = B^a \mod p = 19^6 \mod 23 = 2$гҖӮ
    • Bob и®Ўз®—$S = A^b \mod p = 8^{15} \mod 23 = 2$гҖӮ
    • еҸҢж–№еҫ—еҲ°зӣёеҗҢзҡ„е…ұдә«еҜҶй’Ҙ$S = 2$гҖӮ

Diffie-Hellman зҡ„е®үе…ЁжҖ§

  1. зҰ»ж•ЈеҜ№ж•°й—®йўҳпјҡ
    • ж”»еҮ»иҖ…еҚідҪҝзҹҘйҒ“$p, g, A, B$пјҢд№ҹж— жі•иҪ»жҳ“и®Ўз®—еҮә$a$жҲ–$b$пјҢеӣ дёәи®Ўз®—зҰ»ж•ЈеҜ№ж•°пјҲеҚід»Һ$A = g^a \mod p$дёӯжұӮеҮә$a$пјүеңЁи®Ўз®—дёҠжҳҜдёҚеҸҜиЎҢзҡ„гҖӮ
  2. дёӯй—ҙдәәж”»еҮ»пјҡ
    • Diffie-Hellman жң¬иә«дёҚжҸҗдҫӣиә«д»ҪйӘҢиҜҒпјҢеӣ жӯӨе®№жҳ“еҸ—еҲ°дёӯй—ҙдәәж”»еҮ»пјҲMan-in-the-Middle AttackпјүгҖӮ
    • дёәдәҶи§ЈеҶіиҝҷдёӘй—®йўҳпјҢйҖҡеёёйңҖиҰҒз»“еҗҲж•°еӯ—зӯҫеҗҚжҲ–е…¶д»–иә«д»ҪйӘҢиҜҒжңәеҲ¶гҖӮ

Diffie-Hellman зҡ„еә”з”Ё

  1. TLS/SSLпјҡ
    • Diffie-Hellman жҳҜ TLS/SSL еҚҸи®®дёӯеҜҶй’ҘдәӨжҚўзҡ„дёҖз§Қеёёз”Ёж–№жі•пјҲеҰӮ DHE е’Ң ECDHEпјүгҖӮ
  2. VPNпјҡ
    • и®ёеӨҡ VPN еҚҸи®®пјҲеҰӮ IPsecпјүдҪҝз”Ё Diffie-Hellman жқҘе»әз«Ӣе®үе…Ёзҡ„йҖҡдҝЎйҖҡйҒ“гҖӮ
  3. SSHпјҡ
    • SSH еҚҸи®®д№ҹдҪҝз”Ё Diffie-Hellman жқҘеҚҸе•ҶдјҡиҜқеҜҶй’ҘгҖӮ

Diffie-Hellman зҡ„еҸҳз§Қ

  1. жӨӯеңҶжӣІзәҝ Diffie-HellmanпјҲECDHпјүпјҡ
    • еҹәдәҺжӨӯеңҶжӣІзәҝеҜҶз ҒеӯҰпјҢжҸҗдҫӣжӣҙй«ҳзҡ„е®үе…ЁжҖ§е’Ңжӣҙе°Ҹзҡ„еҜҶй’Ҙй•ҝеәҰгҖӮ
  2. йқҷжҖҒ Diffie-Hellmanпјҡ
    • дҪҝз”Ёеӣәе®ҡзҡ„е…¬й’Ҙе’Ңз§Ғй’ҘпјҢйҖӮз”ЁдәҺй•ҝжңҹйҖҡдҝЎгҖӮ
